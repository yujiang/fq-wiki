# 南姗

<CharImage :icon="2200" size="lg"/>

杏花村医生。

早年间，百草门前代门主 孙思邈 行医途经杏花村，
见其心性纯良、手稳眼细，
遂传她数手医术，
并留下一本亲著的《百草经》。

医术虽不敢言精深，
却胜在耐心与细致。
村中老幼有恙，多半先想到她。

性情温和，见不得生灵受苦。
对小动物尤为怜惜，
——不能吃兔兔，是她的底线。

她自己并不自觉，
但在真正的医者眼中，
她欠缺的并非手段，
而是面对生死时的决断。
