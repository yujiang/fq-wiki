# 苏真真

<CharImage :icon="2073" size="lg"/>

杏花村布店老板。

平日里替村人裁衣缝补，
针脚细密，样式朴素，
做的是最寻常的营生。

闲时最爱看的，
是 云中阁主 的话本。
字里行间的江湖与风月，
让她在布帛与针线之外，
偷偷向往着另一种人生。

她对云中阁主其人并无太多了解，
只是借着话本，
在想象中勾勒出一个
“与自己毫不相干、却极其遥远”的世界。

### 后续剧情

* 绣坊传入（九曲连环坞支线）

在九曲连环坞主线中，
她将作为一条支线人物介入。

真相逐渐揭开——
她的生母，
原是京中绣坊一等一的绣女。

因一次绣品失误，
将 四爪之龙误绣为五爪，
被定为欺君之罪，
族人尽诛。

事发之前，
尚在襁褓中的她被暗中送离，
最终辗转落脚于杏花村，
才得以存活。
