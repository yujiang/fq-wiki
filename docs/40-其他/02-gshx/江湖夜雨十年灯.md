# 江湖夜雨十年灯  我凭一剑候君归

<audio src="/music/mp3/fengqi.mp3" controls></audio>

红尘相守，是何等之难！  
不久之前，老东家邀我看了一场他编导的《曲云传》；旁边的女子跟台上的慕容追风，竟同时道出了这句台词。之后我们去大排档喝酒，后来才发现，我们谈起的都是二十年前的事情。  
那再多的往事，也不过是两个字--青春。  
青春放飞梦想，曾许人间一流；  
如今华发尽染，不道人间怨尤。  

盖世豪侠的重开了在QQ群里首先传开了，我也看到了那么多熟悉的ID开始跳跃闪动；  
匆匆多年，多少人我们都以为散落天涯再无交集，再看他笑着转身竟湿了眼眶。  
时光斑驳，还记得当初的承诺和相守？  
铁甲依然在。  
那剑呢？  
剑，也在。  

“老俞你还记得我吗？我是xxx啊。”  
哈哈，大家在担心我的记忆，确实我已经不是那个拔剑而战通宵达旦的青年了；
而是思过崖上的老头，我有一剑，等你们来取吧。   
“老俞，我定当归来。”  
“备酒以待。”  

红尘相守，是何等之难。  
而能再聚首江湖——  
又是何等之幸。
