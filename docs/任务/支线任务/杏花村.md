# 🌸 杏花村 - 支线任务

## 【赊欠的酒钱】

**接任务**：醉霄楼 · 黄四娘  
**任务流程**：  
1. 与黄四娘对话，得知有人赊欠酒钱未还。  
2. 前往张富贵处 `[92,287]` 追讨酒钱。  
3. 前往杂货店 `[104,206]` 购买 **火折子**。  
4. 再次返回张富贵 `[92,287]`。  
5. 使用轻功跳上山洞 `[198,192]`。  
6. 在洞内开启箱子 `[1]`、`[2]`、`[3]` —— 开箱前需用火折子点亮火把。  
7. 返回醉霄楼向黄四娘复命。  

**任务奖励**：  
- 铜钱 ×100  
- 黄四娘好感 +5

**任务奖励**：

- 村民声望 +2

## 【傀戏人】

**任务目标**：找胡秋侠调查傀戏人为何能说话。  
**任务线索**：  
- 傀戏人每天于傍晚在杏花树下演出；  
- 胡秋侠怀疑其体内机关中藏有“传声木”；  
- 若调查后保持沉默，可触发“镜心”隐藏好感事件。  

**任务奖励**：  
- 神秘木偶部件 ×1  
- 胡秋侠好感 +3
